# 🚀 Portfólio de Desenvolvimento – Software + Hardware

Bem-vindo ao meu portfólio!
Este projeto representa minha jornada como desenvolvedor focado na integração entre **software e hardware**, explorando desde fundamentos da web até aplicações práticas conectadas ao mundo físico.

---

## 🧠 Objetivo

Este portfólio foi criado com dois propósitos principais:

* 📚 **Aprendizado contínuo** em desenvolvimento web
* 🔌 **Integração com hardware**, entendendo como o software controla e se comunica com dispositivos físicos
* 💼 **Apresentação profissional** dos meus projetos e evolução como desenvolvedor

---

## 🛠️ Stack Tecnológica

Este projeto foi desenvolvido utilizando tecnologias essenciais da web:

* **HTML5** → Estruturação das páginas
* **CSS3** → Estilização e responsividade (Flexbox e Grid)
* **JavaScript (ES6+)** → Interatividade e lógica de programação

---

### 📁 Projetos Web

## 🔗 Integração Software + Hardware

Um dos focos principais deste portfólio é explorar como o JavaScript pode interagir com o mundo físico.
